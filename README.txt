Open-sourse проект в инкубаторе  geekbrains.ru https://geekbrains.ru/incubator

Идея проекта 
Простая игра «рогалик» c процедурной генерацией уровней на JavaScript и HTML5 canvas. 
Графика по свободной лицензии типа GPL допустим с сайта http://opengameart.org/. 
Сборка уровней допустим с помощью редактора тайловых карт Tiled, с сохранением в формат JSON. 
HTTP запросы к серверу через AJAX c помощью XMLHttpRequest и разбор JSON файла с помощью метода JSON.parse. 
отрисовка слоев графики и объектов игры на холсте. Реализация логики поведения объектов и взаимодействия с пользователем.

Конечная цель
Задача максимум проекта в отработке технологии создания игр с процедурной генерацией уровней.
Задача минимум обменятся опытом с заинтересованными в этой теме.

Целевая аудитория
Программисты-разработчики

Стадия проекта
Техническое задание в общих чертах есть.
